### Spinner Customized

<CodeBlock
    code={`

	`}
    live
/>

## Props

`width`

**Description**

It is used to set the width of the spinner.

**Values**

String with valid width.

`height`

**Description**

It is used to set the height of the spinner.

**Values**

String with valid height.

`color`

**Description**

It is used to set the color of the spinner.

**Values**

primary | secondary | accent | error | warning | success
