# Alert

---

## Import

<CodeBlock
    code={`
import { Alert } from '@zeal-ui/core';
	`}
/>

## Usage

<CodeBlock
    code={`
<Container type="col">
	<Alert type="error">
		<Text>A problem has occurred try again later</Text>
	</Alert>
	<Alert type="warning">
		<Text>Your network connection is unstable</Text>
	</Alert>
	<Alert type="success">
		<Text>Your message was sent successfully</Text>
	</Alert>
</Container>
	`}
    live
/>

## Props

`type`

**Description**

It is used to set the type of the alert.

**Values**

error | warning | success

`width`

**Description**

It is used to set the width of the alert.

**Values**

String with valid width.

`height`

**Description**

It is used to set the height of the alert.

**Values**

String with valid height.
