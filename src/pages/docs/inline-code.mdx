## Props

`width`

**Description**

It is used to set the width of inline code.

**Values**

String with valid width.

`height`

**Description**

It is used to set the height of the inline code.

**Values**

String with valid height.

`color`

**Description**

It is used to set the color of inline code.

**Values**

primary | secondary | accent | error | warning | success
