## Props

### Avatar

<InlineCode>src</InlineCode>

**Description**

It is used to set the source of the image for the avatar.

**Values**

Path to the image.

<Divider />

<InlineCode>alt</InlineCode>

**Description**

It is used to set the alternate text when image is not available.

**Values**

Text describing the image.

<Divider />

<InlineCode>width</InlineCode>

**Description**

It is used to set the width of the avatar.

**Values**

String with valid width.

<Divider />

<InlineCode>height</InlineCode>

**Description**

It is used to set the height of the avatar.

**Values**

String with valid height.

<Divider />

### Avatar Badge

<InlineCode>color</InlineCode>

**Description**

It is used to set the color of the avatar badge.

**Values**

"gray" | "red" | "yellow" | "blue" | "orange" | "green" | "purple" | "pink"
