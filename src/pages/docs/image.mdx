## Props

`src`

**Description**

It is used to set the source of the image.

**Values**

Path to the image.

`alt`

**Description**

It is used to set the alternate text when image is not available.

**Values**

Text describing the image.

`width`

**Description**

It is used to set the width of the image.

**Values**

String with valid width.

`height`

**Description**

It is used to set the height of the image.

**Values**

String with valid height.

`type`

**Description**

It is used to set the type of the image.

**Values**

row | col | round
