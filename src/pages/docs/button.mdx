## Props

<InlineCode>width</InlineCode>

**Description**

It is used to set the width of the button.

**Values**

String with valid width.

<Divider />

<InlineCode>height</InlineCode>

**Description**

It is used to set the height of the button.

**Values**

String with valid height.

<Divider />

<InlineCode>color</InlineCode>

**Description**

It is used to set the color of the button.

**Values**

"gray" | "red" | "yellow" | "blue" | "orange" | "green" | "purple" | "pink"

<Divider />

<InlineCode>type</InlineCode>

**Description**

It is used to set the type of the button.

**Values**

"floating"

<Divider />

<InlineCode>top | right | bottom | left</InlineCode>

**Description**

It is used to set the position of the button when the type is set to "floating".

**Values**

String with valid position.
