## Props

### List

`type`

**Description**

It is used to set the type of the list.

**Values**

ol | stacked | link

`styleType`

**Description**

It is used to set the type of the style for the list items in the ordered and unordered list.

**Values**

String with valid **list-style-type** value.

### ListItem

`active`

**Description**

It is used to set link as active.

**Values**

Boolean value.
