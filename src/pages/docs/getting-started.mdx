# Getting Started

---

## Installation

Install `Zeal UI` and `styled-components` in your React project directory by executing the following command.

<CodeBlock
    code={`
npm install @zeal-ui/core styled-components
    `}
    language="bash"
/>

## Setting up the Provider

Before working with Zeal UI, you need to set up the `ZealProvider` at the root of your application.

<CodeBlock
    code={`
import React from 'react';
import { ZealProvider } from '@zeal-ui/core';
\r
function App ({ Component }) {
    return (
        <ZealProvider>
            <Component />
        </ZealProvider>
    )
};
    `}
/>
