# Grid

-   A Grid is a two dimentional layout which is used to arrange items horizontally and vertically.

---

## Import

<CodeBlock
    code={`
import { Grid } from '@zeal-ui/core';
  `}
/>

## Usage

<CodeBlock
    code={`
<Grid row={2} col={3}>
  <Badge width="4rem">Item 1</Badge>
  <Badge width="4rem">Item 2</Badge>
  <Badge width="4rem">Item 3</Badge>
  <Badge width="4rem">Item 4</Badge>
  <Badge width="4rem">Item 5</Badge>
  <Badge width="4rem">Item 6</Badge>
</Grid>
  `}
    live
/>

## Props

`row`

**Description**

It is used to set the number of rows in the grid.

**Values**

Valid number.

`col`

**Description**

It is used to set the number of columns in the grid.

**Values**

Valid number.
